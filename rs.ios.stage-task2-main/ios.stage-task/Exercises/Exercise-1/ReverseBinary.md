# Reverse Binary

Function accepts unsigned 8 bits integer and returns its binary reverse (padded to 8 bits).

### Example 1
```
n = 3
binary representation = 0000 0011
reversed = 1100 0000
result = 192
```

### Example 2
```
n = 56
binary representation = 0011 1000
reversed = 0001 1100
result = 28
```
